Illinois Campus Recreation //
Staff Intranet

Customized Wordpress Template

Screen Grabs (password protected site)

Once I converted the public web site to Wordpress, the staff intranet was next. I changed the colors and some basic template settings to create enough of a difference to be able to tell the two web sites apart quickly. Wordpress has made it easy for students and staff to use and update the intranet regularly and efficiently. Creating the same website enables me to train students to start making edits on the staff intranet. Seeing the students that excel in the non-public site allows me to offer hand-pick students to help out with the current web site edits on the public site.
